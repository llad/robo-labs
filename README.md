# robo-labs
Exploration of robotics concepts

GitHub Pages root: [llad.github.io/robo-labs](https://llad.github.io/robo-labs/)

## Single-page apps
Each app below runs as a standalone GitHub Pages HTML file.

- **A* Navigation Visualizer** — Visualizes A* pathfinding with interactive cues.
  - GitHub Pages: [Navigation A* Visualizer](https://llad.github.io/robo-labs/navigation-a-star-visualize.html)
  - Source: [navigation-a-star-visualize.html](navigation-a-star-visualize.html)
- **SLAM Explorer** — Interactive SLAM concepts and visualizations.
  - GitHub Pages: [SLAM Explorer](https://llad.github.io/robo-labs/SLAM.html)
  - Source: [SLAM.html](SLAM.html)
- **Virtual Sensor Algorithms** — Experiments with virtual sensor fusion and signal processing.
  - GitHub Pages: [Virtual Sensor Algorithms](https://llad.github.io/robo-labs/virtual-sensor-algorithms.html)
  - Source: [virtual-sensor-algorithms.html](virtual-sensor-algorithms.html)
